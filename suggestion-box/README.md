# Suggestion Box

What should we build? What's missing? What would make this better?

Drop a note here. No format required — a sentence, a paragraph, a rant. If you think it, write it down.

This is for feedback on the project itself — the repo, the tools, the workflow, the creative direction, anything. If it's a creative idea (song, chapter, art), put it in `content/ideas/` instead.
